# On the density of Kravitz sets

This repository contains a formalized proof of **Theorem 1** from the paper  
**“On the density of Kravitz sets”** by Vsevolod Lev, Máté Matolcsi, Péter Pál Pach, and Dániel Varga.

Paper: <https://arxiv.org/abs/2510.16522>

### Theorem 1

If \(p > 3\) is a prime and \(A \subseteq \mathbb{F}_p\) is a subset with

\[
|A| > \frac{2}{7}p,
\]

then

\[
A + A - 2A = \mathbb{F}_p.
\]
